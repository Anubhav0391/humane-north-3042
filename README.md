# About Project: <br/>
BigBasket is India's biggest e-grocer. It offers tens of thousands of different products, including fruits and vegetables, meats, spices, toiletries, baby products and pet food.
<br/>

2. CSS<br/>
3. JavaScript<br/>
4. ReactJS<br/>
5. ReduxJS<br/>
6. ChakraUI<br/>
7. JSON Server<br/>
